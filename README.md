# Cleanse the Corrupted Garden - Stylized Nature Challenge

My submission for
the [ThreeJS Journey Course's June 2026 Challenge](https://threejs-journey.com/challenges/024-stylized-nature).

## Installation

Install with npm:

```bash
  cd stylized-nature-challenge
  npm install
  npm run dev
```

Open at http://localhost:5173/ or http://localhost:5173/#debug.
